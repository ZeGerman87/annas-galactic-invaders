# Drop sprites here

Place your PNGs in this folder using the exact filenames from `../../docs/ASSETS.md`:

```
player-ship.png
enemy-01.png  enemy-02.png  enemy-03.png  enemy-04.png
ufo.png
boss-01.png  boss-02.png  boss-03.png  boss-04.png  boss-05.png
shot-player.png  shot-enemy.png  shot-boss.png
powerup-rapid.png  powerup-spread.png  powerup-shield.png  powerup-life.png
```

Optional: `background.png`, `logo.png`, `enemy-0X-b.png` (second animation frame).

Anything missing falls back to placeholder art, so the game stays playable while assets arrive.
